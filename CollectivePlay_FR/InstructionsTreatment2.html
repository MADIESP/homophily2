Partie 3 - Instructions - Jeu Collectif


Votre classement définira votre partenaire pour les 6 périodes à venir.

A chaque période:

Vous verrez le même tableau rempli de 0 et de 1.

Vous aurez chacun 30 secondes pour compter tous les zéros.

Votre réponse ou celle de votre partenaire sera choisie au hasard comme réponse de l'équipe.

Si cette réponse est correcte, vous gagnerez tous les deux 1 point; sinon, vous gagnerez tous les deux 0 point.

Ensuite, vous recevrez un feedback sur la performance de l'équipe et vous pourrez échanger un message avec votre partenaire.

Si vous avez des questions, veuillez lever la main pour que nous puissions vous aider.

Nous allons maintenant vous présenter un exemple pour illustrer les différents types de feedback et les différents messages qu’il vous sera possible d’envoyer.

Cliquez sur le bouton pour lancer la suite de l'expérience.
