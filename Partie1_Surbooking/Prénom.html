Partie 1- Sélection de votre prénom


Nous à allons à présent vous demander de sélectionner un prénom fictif parmi une liste de prénoms. Ce prénom vous sera attribué durant toute l’expérience.

Tous les participants verront la même liste de prénoms. Vous choisirez donc votre prénom en même temps que les autres joueurs.

Deux participants ne peuvent pas avoir le même prénom: si un prénom a déjà été sélectionné par un joueur, nous vous demanderons d’en sélectionner un autre.

Si vous avez des questions, veuillez lever la main pour que nous puissions vous aider.

Cliquez sur le bouton pour commencer.
