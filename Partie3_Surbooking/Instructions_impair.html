Partie 3 - Instructions - Jeu Collectif


Dans cette partie, vous allez être sélectionné de manière aléatoire pour soit continuer à réaliser une tache seul, soit être apparié avec un autre participant.

Si vous êtes sélectionné pour rester seul, vous allez réaliser la même tâche que dans la partie 2, mais durant 6 périodes et avec du feedback :

A chaque période :

Vous découvrirez un tableau rempli de 0 et de 1.

Vous aurez 30 secondes pour compter tous les zéros.

Si votre réponse est correct, vous gagnerez un point.

Ensuite, vous recevrez un feedback sur votre performance.

Si vous êtes sélectionné pour être apparié avec un autre participant, vous allez réaliser une tache commune avec ce participant.

A chaque période:

Vous verrez le même tableau rempli de 0 et de 1.

Vous aurez chacun 30 secondes pour compter tous les zéros.

Votre réponse ou celle de votre partenaire sera choisie au hasard comme réponse de l'équipe.

Si cette réponse est correcte, vous gagnerez tous les deux 1 point; sinon, vous gagnerez tous les deux 0 point.

Ensuite, vous recevrez un feedback sur la performance de l'équipe.

Si vous avez des questions, veuillez lever la main pour que nous puissions vous aider.

Nous allons maintenant vous présenter un exemple pour illustrer les différents types de feedback.
